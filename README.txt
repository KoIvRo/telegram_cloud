## Программа рисует облако из слов часто используемых в переписке в tg
### Инструкция:
* Скачайте проект и Установите зависимости
`pip install -r requirements.txt`
* Экспортируйте переписку из телеграмма
1. В декстопной версии мессенджера откройте чат
2. Справа сверху экспортировать историю
3. В открывшемся окне
4. Формат json
* После загрузки копируем файл result.json в папку с файлом project.py из репозитория
* Запускаем project.py
