Программа рисует облако из слов часто используемых в переписке в tg
Инструкция:
1. Скачайте проект и Установите зависимости
pip install -r requirements.txt
2. Экспортируйте переписку из телеграмма
В декстопной версии мессенджера откройте чат
Справа сверху экспортировать историю
В открывшемся окне
Формат json
3. После загрузки копируем файл result.json в папку с файлом project.py из репозитория
4. Запускаем project.py
